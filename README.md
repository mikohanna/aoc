# Advent of Code 🎄

A collection of my Python solutions for the [Advent of Code](https://adventofcode.com/) puzzles.

## 🌟 Structure & Progress
- Organized into separate folders by **Year**.
- I only upload days where I managed to grab both stars (**2 stars ⭐⭐**).
- None of the years are fully completed, as I solve these purely for fun and logic practice in my free time.

## 🍝 A Disclaimer on Code Quality & Consistency
Many of these solutions were written when I was actively learning Python, so this repository is essentially a personal scratchpad. Because of this:
- **Style is inconsistent:** Some scripts output clean, formatted text, while others just print the raw answer.
- **Scratchpad remnants:** You might find commented-out `print()` statements, unrefined variable names, and sub-optimal solutions.

This repo is simply a record of my algorithmic thinking, logic, and how I approached problem-solving while getting a grip on Python.
